## 5. 표(Table) 삽입 시 주의

### 5.1 2단 조판에서 표 크기

- **단 폭 ≈ 80mm** (190 - 20 - 20 - 5 = 145mm, 나누기 2 = 약 72.5mm + 여유)
- 표 1, 5: 열이 적으므로 **1단 폭**에 들어감
- **표 2, 3, 4, 4-1**: 열이 많으므로 **2단 통합(단 합치기)** 필요

### 5.2 HWP에서 2단 통합 표 넣기

1. 표 삽입할 위치에 커서
2. **쪽 → 다단 설정 → 1단**으로 변경
3. 표 삽입
4. 표 아래에서 다시 **쪽 → 다단 설정 → 2단**으로 복귀

### 5.3 Word에서 2단 통합 표 넣기

1. 표 직전에 **구역 나누기(섹션 나누기)** → 1단으로 변경
2. 표 삽입
3. 표 직후에 다시 구역 나누기 → 2단 복귀

### 5.4 표 캡션 위치

- 한국 학회 관례: **표 위에 캡션** (예: "표 1. PPO 학습 하이퍼파라미터")
- poster_final.md의 "**표 1.**" 형식 그대로 사용

## 8. 분량 조절 전략

**목표: 3~5페이지 (190×260mm, 2단)**

### 예상 분량

- 현재 본문 텍스트: ~3,500자 → **약 2.5페이지** (2단 기준)
- 표 6개 (표 1~5, 표 4-1): 약 1~1.5페이지
- 그림 5장: 약 1~1.5페이지
- 참고문헌: 약 0.3페이지

→ **총 약 5~5.5페이지 예상** (약간 초과 가능)

### 분량 초과 시 조절 방법 (우선순위)

1. 그림 크기 축소 (2단 통합 → 1단 폭으로)
2. 표 4-1 (baseline 비교)을 본문 서술로 대체
3. 후보 그래프(graph3, 4, 7, 8) 삽입하지 않음
4. 3.3 VsMelee 세부 설명 축소

### 분량 부족 시 (3페이지 미만)

1. graph8 (VsMelee 승률·보상 추이) 추가 삽입
2. graph4 (VsHybrid 명중률·전투시간) 추가 삽입
3. 게임 스크린샷 추가

## 9. 작업 순서 권장

```
1. 양식 HWP 다운로드 → 열기
   https://ki-it.or.kr/file/download/12581

2. 제목/저자/소속/이메일 입력 (양식 상단)

3. 요약 → Abstract → Keywords 복사

4. 본문 Ⅰ~Ⅳ 복사 (서식 자동 적용됨)

5. 표 삽입 (표 1~5, 표 4-1)
   → 큰 표는 2단 통합으로

6. 그림 삽입 (그림 1~5)
   → 캡션 붙이기
   → 필요시 dpi=300으로 그래프 재생성

7. 참고문헌 복사

8. 전체 페이지 수 확인 (3~5p)
   → 초과 시 그림 축소 / 부족 시 후보 그래프 추가

9. 최종 PDF 변환 후 제출
```
